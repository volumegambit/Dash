Logo Architecture

Flexible mark system: icon + wordmark, separable for different contexts.

⚡
dashsquad
Primary lockup
Icon + lowercase wordmark. Used on homepage, marketing, docs. The icon should be a custom mark (not literally a lightning bolt — placeholder here).
⚡
dashsquad
Compact
Navbar, app header, small spaces. Same lockup, scaled down.
⚡
Icon only
Favicon, app icon, social avatars, loading states. Must be recognizable at 16×16.

Naming: "dashsquad" — all lowercase, one word. Never "Dash Squad" or "DashSquad" in the wordmark. Legal entity: Dash AI Labs.

Graphic Signatures

Recurring visual patterns that make DashSquad instantly recognizable.

Bold 2px Strokes

Hard black borders on cards, inputs, and images. No soft shadows, no gradients on containers. Confident and graphic.

NOW IN ALPHA

Status Badges

Black pill badges with green status dot and mono text. Used for status indicators, announcements, tags.

FEATURES

Mono Section Labels

JetBrains Mono, all-caps, wide tracking, Power Orange. Precedes every section headline. Technical rhythm.

Design principle: Square corners on functional elements (cards, inputs, badges). No border-radius except the logo icon. This creates a graphic, editorial feel that separates DashSquad from the rounded-corner SaaS crowd.
