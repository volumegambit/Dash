Dash helps you safely deploy autonomous "claw" agents on your own machine or private cloud. Connect agents to the LLM provider you choose, integrate them with messaging channels like Telegram, and keep your data private and under your control.

We believe helping people deploy their own personal or work agents is the best way for everyone to experiment and understand how AI can make their lives better. This has to be done safely — Dash provides secure defaults, proper secrets management, and isolation for each agent.

## Architecture

Five packages organized in a layered dependency graph:

```
server → channels → agent → llm
              tui ↗
```

| Package | Role |
|---------|------|
| `@dash/llm` | LLM provider abstraction and Anthropic streaming |
| `@dash/agent` | Agent orchestration, sessions, and tool execution |
| `@dash/channels` | Channel adapters (Telegram) and message routing |
| `@dash/server` | Entry point, config loading, and gateway wiring |
| `@dash/tui` | Terminal UI / CLI entry point |

## Quick start

<Steps>
  <Step title="Install dependencies">
    ```bash
    git clone <repo-url> Dash
    cd Dash
    npm install && npm run build
    ```
  </Step>
  <Step title="Configure credentials">
    ```bash
    cp .env.example .env
    ```
    Add your `ANTHROPIC_API_KEY` to the `.env` file.
  </Step>
  <Step title="Run the TUI">
    ```bash
    npm run dev --workspace=packages/tui
    ```
  </Step>
</Steps>

See [getting started](/getting-started) for the full setup guide.
